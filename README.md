# 🔢 NumPy Exercises :

A collection of beginner-friendly Jupyter Notebooks covering core NumPy concepts with hands-on code examples. Great for anyone learning data manipulation with Python.

---

## 📁 Notebooks

| Notebook | Topics Covered |
|---|---|
| `Adding and Removing.ipynb` | `np.append`, `np.insert`, `np.delete` |
| `Aggregating Function.ipynb` | `sum`, `min`, `max`, `mean`, `cumsum`, `cumprod` |
| `Inspecting an Array.ipynb` | `shape`, `size`, `dtype`, `astype` |
| `Join and Split.ipynb` | `concatenate`, `hstack`, `vstack`, `array_split` |
| `Mathematical Operations.ipynb` | `add`, `subtract`, `multiply`, `divide`, `power`, `sqrt` |
| `Search-Sort-Filter.ipynb` | `sort`, `where` |
| `Statistical Function.ipynb` | `mean`, `median`, `mode`, `std`, `var`, `corrcoef` |

---

## 🚀 Getting Started

### Prerequisites

Make sure you have the following installed:

- Python 3.x
- NumPy
- Jupyter Notebook or JupyterLab

### Installation
```bash
# Clone the repository
git clone https://github.com/your-username/Numpy-Exercise.git
cd Numpy-Exercise

# Install dependencies
pip install numpy jupyter

# Launch Jupyter
jupyter notebook
```

## 📚 Topics Overview

- **Array Creation & Inspection** — Build arrays and explore their properties like shape, size, and data types.
- **Mathematical Operations** — Perform element-wise arithmetic and apply functions like `sqrt` and `power`.
- **Aggregating Functions** — Summarize arrays using sum, mean, cumulative products, and more.
- **Statistical Functions** — Analyze data with standard deviation, variance, and correlation.
- **Join & Split** — Combine and slice arrays along different axes.
- **Adding & Removing** — Dynamically modify array contents.
- **Search, Sort & Filter** — Find and organize array elements efficiently.
